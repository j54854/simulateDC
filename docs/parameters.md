# シミュレーションパラメータ一覧

`create_simulator` の `params` 引数に辞書で渡すことでデフォルト値を変更できる。
指定しなかったパラメータは `Simulation_Environment` のクラス変数のデフォルト値が使われる。

```python
params = {
    'SEED': 42,
    'PJOB': 500,
    'VEHICLE': 40,
}
env = create_simulator(params=params)
```

---

## ジョブ・制御パラメータ

| パラメータ | デフォルト | 単位 | 説明 |
|-----------|-----------|------|------|
| `SEED` | 1234 | — | 乱数シード |
| `PJOB` | 1000 | 件 | シミュレートするピッキングジョブ総数 |
| `OPENABLE` | 5 | 個 | 同時に開封可能なバケット数（バケット割付け候補の上限） |
| `RELEASABLE` | 50 | 件 | システム内に同時投入できる搬送ジョブ数の上限 |
| `REPEATABLE` | 3 | 回 | 許容する荷卸し失敗回数（これを超えると置場に戻される） |

---

## 設備規模パラメータ

| パラメータ | デフォルト | 単位 | 説明 |
|-----------|-----------|------|------|
| `AISLE` | 7 | 列 | 倉庫の系列数 |
| `FLOOR` | 6 | 階 | 倉庫の階数 |
| `ROW` | 45 | 段 | 各系列の置場数（0〜ROW-1 が格納スロット，ROW 番目が台車待機ノード） |
| `PICKER` | 5 | 台 | ピッキング作業場数 |
| `VEHICLE` | 30 | 台 | ループの台車数 |
| `LOOP` | 100 | ノード | ループのノード数 |
| `ITEM` | 100 | 種 | アイテム種類数 |

> **注意**: `AISLE_SEG`・`PICKER_SEG`・`CONV_LEN`・`CONV_DIM` はループ・コンベヤの接続トポロジを定義しており，設備規模を変更する場合は合わせて調整が必要。通常は変更しない。

---

## タイムパラメータ

単位はすべて **秒**。

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `MT_SHTL` | 0.2 | シャトル台車のノード間移動時間（1ノード = 1置場分の距離） |
| `LT_SHTL` | 5 | シャトル台車との積替え時間（乗降各1回） |
| `MT_LIFT` | 0.2 | リフト台車のノード間移動時間（1ノード = 1階分の距離） |
| `LT_LIFT` | 5 | リフト台車との積替え時間 |
| `MT_LOOP` | 0.2 | ループ台車のノード間移動時間 |
| `LT_LOOP` | 1 | ループ台車との積替え時間 |
| `MT_CONV` | 1 | コンベヤのノード間移動時間（作業場への出入りも同値） |

---

## ピッキング作業時間

ピッキング作業時間は `T_PICK` から `P_PICK` の確率分布に従いランダムに決定される。

| パラメータ | デフォルト | 説明 |
|-----------|-----------|------|
| `T_PICK` | `[6, 9, 12, 24, 36, 48, 60]` | 作業時間の候補値（秒） |
| `P_PICK` | `[0.4, 0.4, 0.1, 0.04, 0.03, 0.02, 0.01]` | 対応する確率分布（合計 1.0） |
